CSS Styling Cheatsheet

CSS Properties

Property Value Description
color: "color name"; RGB; HSL; Hex Code; Changes the font color.
background-color: "color name"; RGB; HSL; Hex Code; Changes the background color of the element.
background-image: url("img link"); Changes the background of the element to the selected image.
border: ("width", "style", "color"); Specifies the style, width, and color of an element's border.
position:
  1. static;
  2. absolute;
  3. relative;
  4. fixed;
  5. sticky;
  1. Positioned elements are not affected by the top, bottom, left, and right properties
  2. Positioned relative to the nearest positioned ancestor
  3. Positioned relative to its normal position. Can be off-set left, right, top or bottom by pixels
  4. Positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled
  5. Positioned based on the user's scroll position
font-family: name; or "name name"; Property to specify the font of a text. If two words, must be in quotation marks
font-style:
  1. normal;
  2. italic;
  3. oblique;
  1. Text is normally shown
  2. Text is in italics
  3. The text is "leaning" (oblique is very similar to italic, but less supported)
font-size: "#px"; Sets the size of the text
height:/width: "#px"; Used to set the height and width of an element
max-height:/max-width: "#px"; Used to set the max-height and max-width of an element
z-index: "#"; Specifies the stack order of an element
padding: "#px"; Creates space around an element's content, inside of any defined borders.
margin: "#px"; Creates space around elements, outside of any defined borders. Can be set to specify top, bottom, left, or right.
overflow:
  1. visible;
  2. hidden;
  3. scroll;
  4. auto;
  1. Default. The overflow is not clipped.
  2. The overflow is clipped, and the rest of the content will be invisible
  3. The overflow is clipped, and a scrollbar is added to see the rest of the content
  4. Similar to scroll, but it adds scrollbars only when necessary
